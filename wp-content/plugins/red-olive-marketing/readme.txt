== Installation ==
**VIA THE WORDPRESS BACKEND**
1. Download the plugin
2. In the WordPress backend click Plugins > Add New > Upload Plugin > Click 'Choose File' and select the plugin zip file
3. Click 'Install Now'
4. Click 'Activate Plugin'
5. Hover over the Red Olive logo and name in the WordPress backend sidebar and click 'Marketing Free'
6. Set up your tags and get to marketing the site

**VIA FTP**
1. Download the plugin
2. Extract the zip file
3. Upload the `ro-marketing` folder to the `/wp-content/plugins/` directory of your website
4. Activate the plugin through in your WordPress backend (WordPress > Plugins > Red Olive Marketing > Activate
5. Hover over the Red Olive logo and name in the WordPress backend sidebar and click 'Marketing Free'
6. Set up your tags and get to marketing the site

1. Plugin Configuration / Settings Location

3. Call Tracking Tab | Add the CallTrackingMetrics Tag. For Discounted Rates Sign Up via the Link in the Plugin for Discounted Rates of the Dynamic Phone Number Insertion Service.
4. CRO/Conversion Rate Optimization Tab | Add the Google Analytics Content Experiments Tag, Google Optimize Tag, as well as the Crazy Egg Tag. Upgrade to RO Marketing Pro to Add the Content Experiments Tag per Device Type (Desktop, Tablet, Mobile, Tablet and Phone, or All Devices.) RO Marketing Pro also has an On-site Text Variation Creator for A/B Testing that Adds an Appendage to the URL.
5. Email Tab | Upgrade to RO Marketing Pro to Create an Email Signup Form with a Shortcode to Add Anywhere on your Site with Custom CTA Button Text, Custom Placeholder Text, and a Custom Thank You for Subscribing Message After Submission.
6. Live Chat Tab | Add the LiveChat (livechatinc.com) Setup Tag as well as the Olark Live Chat Setup Tag
7. Local SEO / KML Sitemap Tab | Upgrade to RO Marketing Pro to Create a KML Sitemap (Keyhole Markup Language) to Submit via Google Search Console for Local Search Engine Optimization / Google Maps Results
8. PPC / Pay per Click Tab | Add the Google AdWords Remarketing Tag, and the Bing Ads UET / Universal Event Tracking Tag
9. Promos Tab | Upgrade to RO Marketing Pro to Create a Sitewide Promotional/Announcement Banner that Goes at the Top of your Site with a CTA Button. The Banner can be Persistent/Sticky to the Top and is Responsive / Mobile Friendly
10. QA Tab | Add the Track Duck Tag for Quality Assurance and Bug Tracking Functionality on the Front End of your Site
11. Redirects Tab | Upgrade to RO Marketing Pro to Add 301 Redirects via a Bulk Import or One by One
12. Reviews Tab | Upgrade to RO Marketing Pro to Create and Configure a Reviews Page to Send to your Happy Customers / Clients that Links Out to All of Your Sites you Want Reviews on such as Google, Yelp, Facebook, Angie's List, Houzz, and Any Others
13. Scripts Tab | Add Header Scripts and Footer Scripts to Every Page of your Site
14. Settings Tab | Upgrade to RO Marketing Pro to Stop Comment Spam, Change Image File Names in the Media Library (Even After They've Been Uploaded,) Clear Expired Sessions Automatically to Minimize Server Load / Make Your Site Faster, Force HTTPS Everywhere (You Will Need a SSL Certificate [Secure Sockets Layer Certificate] for This to Work Properly)
15. Feature Request Tab | Want a New Tag Added to RO Marketing? Just Send us a Request for us to Review
